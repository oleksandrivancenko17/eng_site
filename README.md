A production-ready web application for learning English, featuring grammar modules, vocabulary flashcards, and automated testing.

🌍 **Live Demo:** [https://best-devs.studio](https://best-devs.studio)

## 🛠 Tech Stack
* **Backend:** Python 3.13, Django 5.x
* **Frontend:** Django Templates, CSS3, JavaScript
* **Database:** PostgreSQL (Production), Redis
* **Infrastructure:** Docker & Docker Compose
* **Deployment:** Microsoft Azure (Ubuntu 24.04 LTS)
* **Web Server:** Nginx (Reverse Proxy)
* **Security:** Let's Encrypt SSL (Automated HTTPS)

## 🌟 Key Features
* **Interactive Testing:** Automated quiz system with instant validation.
* **Comprehensive Dictionary:** Managed vocabulary with over 4,000 words.
* **Grammar Modules:** Structured learning paths for different proficiency levels.
* **Production Architecture:** Containerized services with a secure gateway and static file optimization.

## 🚀 How to Run Locally
1.  Clone the repository.
2.  Create a `.env` file based on the environment requirements.
3.  Run `docker compose up --build`.
